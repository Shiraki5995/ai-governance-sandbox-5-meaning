This repository demonstrates a simple prototype for handling human-meaning structures using generative AI.
In many AI applications, models process raw text or numerical data.
However, real-world decision making is based on meaningful structures such as:
business processes
organizational roles
decision intentions
operational constraints
This prototype introduces a minimal representation of such structures using Business Process Nodes (CBP Nodes).
Each node represents a meaningful unit of business activity.
The AI model is then asked to reason over these nodes to identify improvements in business performance (in this example, Cash Conversion Cycle).
This project illustrates a key idea:
AI becomes more useful when it reasons over structured meaning, not just raw text.
